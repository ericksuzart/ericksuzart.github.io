ROS package lar_gazebo

In this project the goal is to make a 3D model of the Robotics Laboratory (LAR) of UFBA to be used with Gazebo and ROS.

Project information

  • Category: ROS
  • Project date: 2018
  • Project URL: GitHub

Details

This was originally made for ROS Kinetic, but it was adapted to other versions of ROS(e.g. Melodic, Noetic).

How to use

To use this package, you need to have ROS installed on your computer. After that, you need to create a catkin workspace and clone this repository inside the src folder. After that, you can build the package with your favourite build tool. To run the package, you can use the command roslaunch lar_gazebo lar_world.launch.

How was this model created?

The model was created using the Blender and the Gazebo plugins. The Blender was used to create the 3D model and the Gazebo plugins was used to ajust the model to the Gazebo format using SDF files. The model was created using the measurements and images of the lab aswel as the map generated by the SLAM algorithm. You can see all the details of this project in the GitHub repository.
