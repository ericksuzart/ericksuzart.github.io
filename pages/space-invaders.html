How it works

The game is divided in three phases:

  • Ship Actions: The player can move the spaceship left-right using the buttons.

    The spaceship shoots bullets automatically that move up to hit the enemies.

  • Aliens Actions: The enemies move right/left and shoot bullets randomly that move down to hit the spaceship.

    Depending on the level, the enemies move faster, and the probability of shot rises.

  • Draw the Screen: In this phase the game is drawn on the LCD screen according to the game state.

We use some custom characters to represent the game objects.

With these characters we can draw the game objects on the LCD, that are: space-ship, bullets and enemies(aliens). Each of these objects has a position on screen and especific behavior for each one.

Detail

This game was made as a challenge to gain as award an Arduino Kit.

The game is a simple Space Invaders inspired game using an Arduino UNO, LCD character display, some buttons and a buzzer. The player controls a spaceship and must avoid be hitted by the enemies.

This project was implemented in C++ using the Arduino UNO as a microcontroller. Enjoy it!
